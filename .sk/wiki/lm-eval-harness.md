# EleutherAI LM Evaluation Harness

A unified framework for testing generative language models on many benchmarks.

- **Repo**: https://github.com/EleutherAI/lm-evaluation-harness
- **Key features**: Reproducible prompts, custom metrics, broad benchmark coverage, standardized evaluation

## Relevance to Scidekick

Referenced in the science surface. Scidekick should integrate with the LM Evaluation Harness rather than inventing a private benchmark runner. An adapter is planned in the implementation sequence.
