# AutoScientists

LLM-driven automated scientific research system from MIMS Harvard.

## Key contributions

- Framework for automated hypothesis generation, experiment design, and result analysis
- Tight propose → test → measure loop with LLM agents
- Forms the conceptual basis for Scidekick's experiment modes (loop mode, pipeline mode, team mode)

## Relevance to Scidekick

Referenced in the Scidekick system context as the foundation for the experiment modes. The AutoScientists loop mode (propose → test → measure) directly inspired Scidekick's `sk experiment` workflow design.
