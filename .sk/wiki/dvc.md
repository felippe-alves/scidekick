# DVC (Data Version Control)

Open-source version control system for ML projects — data, models, and pipelines.

- **Docs**: https://doc.dvc.org/
- **Key features**: Content-addressed data versioning, pipeline definition, metric tracking, experiment management

## Relevance to Scidekick

Referenced in the science surface as a system to integrate with. Scidekick should record DVC or content-addressed dataset versions, preprocessing code versions, and split IDs, but should not duplicate DVC's versioning capabilities.
