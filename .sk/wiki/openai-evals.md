# OpenAI Evals

A framework for evaluating LLMs and LLM systems, supporting private custom evals.

- **Repo**: https://github.com/openai/evals
- **Key features**: Custom eval creation, private eval support, standardized eval reporting

## Relevance to Scidekick

Referenced in the science surface. Scidekick should integrate with OpenAI Evals rather than inventing a private benchmark runner. An OpenAI Evals adapter is planned in the implementation sequence (step 6: eval integrations).
